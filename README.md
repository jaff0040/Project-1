# 🏡 Sweet Home Store

A modern and elegant furniture website designed using HTML and CSS.  
This project focuses on clean UI design, smooth layout structure, and user-friendly experience.

---

## ✨ Features

- 🎯 Modern and responsive design
- 🛋️ Furniture showcase with hover effects
- 📦 Offers and services section
- 👥 Team and testimonials sections
- 📰 Recent news cards
- 💰 Pricing table
- 📊 Statistics section with fixed background
- 📱 Clean navigation bar with dropdown menu
- 🌐 Footer with social media icons

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- Font Awesome (icons)
